# Student-Management-System
Student Management System

**Project Purpose**

This project is a simple Student Management System developed using Java. It helps users manage student records by adding, viewing, searching, and deleting student information.

Features

- Add Student
- View All Students
- Search Student by ID
- Delete Student by ID

Technologies Used

- Java
- ArrayList
- Object-Oriented Programming (OOP)

How to Run

1. Download or clone the project.
2. Open the project in any Java IDE (Eclipse, IntelliJ IDEA, VS Code).
3. Compile the Java files.
4. Run the Main.java file.
5. Use the menu options to manage student records.

Example Menu

1. Add Student
2. View Students
3. Search Student
4. Delete Student
5. Exit
